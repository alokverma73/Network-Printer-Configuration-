# Network Printer Configuration

## Project Description
This project demonstrates how to configure a network printer using Cisco Packet Tracer. The objective is to connect a printer to a network so that multiple computers can access and use it through IP addressing.

## Objective
The main goal of this project is to understand network device integration and how a printer can be shared across multiple systems in the same network.

## Tools Used
- Cisco Packet Tracer
- PC / Laptop
- Network Printer
- Switch
- Router

## Network Configuration
- Router IP: 192.168.1.1
- Printer IP: 192.168.1.50
- PCs connected through a switch
- All devices are configured in the same network

## Steps Performed
1. Added router, switch, PCs and printer in Cisco Packet Tracer.
2. Connected all devices using Ethernet cables.
3. Assigned IP addresses to PCs and printer.
4. Configured the printer with a static IP address.
5. Tested connectivity using the ping command.
6. Verified printing access from multiple PCs.

## Result
The network printer was successfully configured and connected to the network. All computers were able to communicate with the printer using its IP address and successfully send print requests.

## Conclusion
This project shows how network printers can be configured and shared among multiple devices, improving resource sharing and network efficiency.
